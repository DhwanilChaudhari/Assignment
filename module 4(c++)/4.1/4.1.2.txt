Q.What is OOP? List OOP concept.
A.Object Oriented Programming (OOP) is a programming paradigm that focuses on the use of objects to represent and manipulate data. 
In OOP, data is encapsulated within objects, and objects are defined by their properties (attributes) and behaviors (methods).
OOP provides several key concepts that enable developers to write modular, reusable, and maintainable code.

List oop concept
1. Class
2. Objects
3. Encapsulation
4. Abstraction
5. Polymorphism
6. Inheritance
7. Dynamic Binding
